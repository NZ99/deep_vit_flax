## Deep Vision Transformer - JAX/Flax

An implementation of <a href="https://arxiv.org/abs/2103.11886">DeepViT: Towards Deeper Vision Transformer</a>, an improved Vision Transformer that avoids attention collapse by exploiting the interactions among different heads.

The official code will be made available in <a href="https://github.com/zhoudaquan/dvit_iccv21">this repo</a>.
